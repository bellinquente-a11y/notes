# pyenv

pyenv lets you install and switch between multiple Python versions.

Useful because different projects may require different Python versions.

Common commands:

```bash
pyenv install 3.12
pyenv global 3.12
pyenv local 3.11
pyenv versions
```

Note:
- global: for entire machine
- local: for the local folder (i.e. project)
